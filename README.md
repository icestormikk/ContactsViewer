## ContactsViewer

ContactsViewer — это мобильное приложение, написанное на Kotlin, которое предоставляет пользователю функциональность для ведения записной книжки контактов.

## Основной функционал
1. Просмотр списка всех контактных записей.
2. Добавление новых контактлв с полной информацией — имя, фамилия, отчество, телефон, адрес и статус (например: друг, брат, отец, врач и т. д.).
3. Редактирование существующих записей.
4. Удаление ненужных контактов.
5. Сортировка и фильтрация контактов по различным полям (имя, фамилия, статус).
6. Поиск контактов по имени, фамилии, отчеству или номеру телефона.
7. Простой и понятный пользовательский интерфейс для работы с контактами

## Языки и инструменты
- Kotlin
- Android SDK
- SQLite (+ Room ORM)

## Установка и запуск
1. Склонируйте репозиторий
```bash
git clone https://github.com/icestormikk/ContactsViewer.git
```
2. Откройте проект
3. Соберите и запустите приложение на эмуляторе или реальном устройстве Android.
